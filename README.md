# Document_checker

**Document_checker** — это система контроля версий документов и учёта изменений в организации. Проект реализован с использованием Django, PostgreSQL и Bootstrap. Система позволяет пользователям загружать документы, отслеживать все версии и изменения, а также видеть разницу между ними.

## Основные возможности

- Регистрация и аутентификация пользователей.
- Создание и хранение документов.
- Автоматическое создание версий документов при изменении.
- Просмотр истории изменений и различий между версиями.
- Управление доступом к документам через роли пользователей.
- Простая и удобная веб-панель на Bootstrap.

## Технологии

- Python 3.12
- Django 4.x
- PostgreSQL 16
- Bootstrap 5
- HTML / CSS / JavaScript
